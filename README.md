# EmailDataExtractor
Dot Net Framework 4.8, Web API

## Project Description

As part of a new integration, we have a need to import data from text we receive via email. The data can be a combination of:
- Embedded blocks or ‘islands’ of XML content
- Marked up fields using opening and closing tags
